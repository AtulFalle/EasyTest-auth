Privacy Policy for Web Test Recorder

Data Collection

Web Test Recorder records the following data during test recording sessions:

  • Screen content of the active tab
  • User interactions (clicks, keyboard inputs)
  • Developer tools information

Data Usage

All recorded data is:

  • Stored locally on your device
  • Never transmitted to external servers
  • Only saved when you explicitly choose to save the recording

Data Control

Users have complete control over their recordings:

  • Start/stop recording at any time
  • Choose where to save recordings
  • Delete recordings directly from their device
